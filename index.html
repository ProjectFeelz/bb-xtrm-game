STUDIO OPS
1. INITIALIZATION
Select a field (ENGINEER, BEATS, or WRITER) to begin. Each session consists of 10 rapid-fire tasks tailored to that discipline.

2. THE CLOCK
You have 30s per task. Audio intensity increases as time runs out. If the timer hits zero, the session is terminated immediately. There is **NO PAUSE**; once you start, the clock never stops.

3. CLOUT (POINTS)
• Base Task: +10 Clout per completion.
• Speed Demon: +5 Bonus if submitted with >15s remaining.
• XP Overdrive: Complete 3 tasks in a row without rerunning for a +25 Combo.

4. RIG MECHANICS
• RERUN: Use this to swap a difficult task. It resets your task timer to 30s but disqualifies you from a Perfect Set.
• SECURITY LOCK: Every task has a 10s cooldown. You cannot "Complete" a task instantly; you must put in the work.

5. GOLD STATUS
To achieve a PERFECT SET and earn a career trophy, you must complete all 10 tasks with 0 Reruns and at least 5 Speed Demon bonuses.
